Blockchain Workshop

Pick an idea:

  • Multisignature wallet

  • Sign and verify documents hashes

  • Proof of existence

Once you decide on a idea to work, make a group and plan how are you going to build it and on which blockchain.

If you decide to build it on Bitcoin you would need to use the bitcore library.

If you decide to build it on Ethereum you would need to use the web3 & open-zeppelin library.
